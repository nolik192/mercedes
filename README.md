# Mercedes-Benz Landing Page

Editorial-luxury brand showcase concept — single scrollable page built with Vite, React, Tailwind CSS, and Motion. Real stock photography of Mercedes-Benz vehicles (S-Class, EQS, G-Class, AMG GT); no backend, no real dealer/lead capture. Not an official Mercedes-Benz property.

See the design spec: [`docs/superpowers/specs/2026-07-11-mercedes-landing-page-design.md`](docs/superpowers/specs/2026-07-11-mercedes-landing-page-design.md)

## Development

```bash
npm install
npm run dev
```
